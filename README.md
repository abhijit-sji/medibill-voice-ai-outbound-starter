# Voice AI — Outbound Calls

ElevenLabs Conversational AI for outbound patient calls. Supports appointment reminders, registration campaigns, and configurable AI agents with custom voices, knowledge bases, and conversation flows.

Version: 1.0.0
Status: stable

## Features
- ElevenLabs AI agents
- Outbound call campaigns
- Custom voice selection
- Knowledge base integration
- Conversation transcripts
- Cost tracking per call
